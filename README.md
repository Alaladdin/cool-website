# Cool website

[Open website](https://coola.netlify.app/)

### Used in this project

- Bootstrap
- JQuery
- Owl carousel
- Gulp
- eslint
- stylelint
- Pug
- sass

### Usage

**Install dependencies**

```shell
npm install
```

**Dev mode**: starts the watcher

```shell
npm run dev
```

**Build**: creates "dist" directory ready to deploy

```shell
npm run build
```

**Clean**: Clean "dist" directory

```shell
npm run clean
```
